# Design System: Onlinein — MASTER

Generated by UI UX Pro Max + brand override dari Plan.md

## Pattern
Feature-Rich Showcase + Trust & Authority
- CTA: Above fold + repeated after social proof
- Sections: Hero → Problem → Services → Pricing → Why → How → Testimonial → FAQ → CTA → Footer

## Style
Trust & Authority (Minimalism/Soft UI Evolution)
- Professional, clean, credentials-driven
- Before/after comparisons, case studies, social proof

## Typography
- Heading: Plus Jakarta Sans (300–700)
- Body: Plus Jakarta Sans
- Google Fonts: https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:ital,opsz,wght@0,8..18,300..800;1,8..18,300..800&display=swap

## Key Effects
- Smooth transitions 150–300ms
- Subtle card hover: shadow-md → shadow-xl
- Gradient text: #4338CA → #06B6D4
- Scroll-reveal via CSS animation

## Anti-patterns (AVOID)
- AI purple/pink gradients
- Harsh animations
- Emoji as icons (use Lucide SVG)
- Playful/cartoonish design
- Hidden pricing or credentials

## Pre-Delivery Checklist
- [ ] No emojis as icons (Lucide SVG only)
- [ ] cursor-pointer on all clickable elements
- [ ] Hover states with smooth transitions (150-300ms)
- [ ] Light mode: text contrast ≥ 4.5:1
- [ ] Focus states visible for keyboard nav
- [ ] prefers-reduced-motion respected
- [ ] Responsive: 375px, 768px, 1024px, 1440px
